# Sampling Methods using Seaborn Tips Dataset

## 📌 Project Overview

This project demonstrates three different sampling methods in Python using the tips dataset from the Seaborn library.

The dataset is treated as a population, and different sampling techniques are used to extract samples.

## Dataset

The dataset used in this project is the **tips dataset** from the Seaborn library.
It contains information about restaurant bills, tips, and customer characteristics.

Main variables include:

* total_bill
* tip
* sex
* smoker
* day
* time
* size

## Sampling Methods

### 1. Simple Random Sampling

Randomly selects observations from the population.

### 2. Systematic Sampling

Selects every k-th observation from the dataset.

### 3. Stratified Sampling

Divides the population into groups (strata) and samples from each group.

## Technologies

* Python
* pandas
* seaborn

## Purpose

The purpose of this project is to demonstrate basic statistical sampling techniques used in data analysis and statistics.

# Seaborn Tips 데이터셋을 이용한 표본추출 방법

## 📌 프로젝트 개요

이 프로젝트는 Seaborn 라이브러리의 tips 데이터셋을 모집단(population)으로 가정하고, 파이썬을 사용하여 세 가지 표본추출 방법을 구현하는 것을 목표로 합니다.

데이터 분석에서 자주 사용되는 기본적인 표본추출 기법을 이해하고 실습하는 것이 목적입니다.

## 데이터셋

본 프로젝트에서는 Seaborn 라이브러리에서 제공하는 **tips 데이터셋**을 사용합니다.

이 데이터셋은 레스토랑에서 발생한 계산 금액과 팁, 고객 특성 등에 대한 정보를 포함하고 있습니다.

주요 변수:

* total_bill (총 계산 금액)
* tip (팁)
* sex (성별)
* smoker (흡연 여부)
* day (요일)
* time (식사 시간)
* size (인원 수)

## 표본추출 방법

### 1. 단순 무작위 추출 (Simple Random Sampling)

모집단에서 무작위로 데이터를 선택하는 방법입니다.

### 2. 체계적 추출 (Systematic Sampling)

데이터에서 일정한 간격(k)마다 하나씩 선택하는 방법입니다.

### 3. 층화 추출 (Stratified Sampling)

모집단을 여러 집단(층)으로 나눈 후 각 집단에서 표본을 추출하는 방법입니다.

## 사용 기술

* Python
* pandas
* seaborn

## 프로젝트 목적

본 프로젝트의 목적은 데이터 분석과 통계에서 사용되는 기본적인 표본추출 방법을 이해하고 파이썬으로 구현하는 것입니다.
